# Security Policy

Do not report vulnerabilities in public GitHub issues.

Use GitHub private vulnerability reporting:

https://github.com/Maillayers/maillayers-client-sdks/security/advisories/new

If that form is unavailable, email `security@maillayers.com` with the subject `MailLayers client SDK security report`.

Include the affected package and version or commit, impact, reproduction steps, and suggested mitigation when available. Remove API keys, customer content, access tokens, private hostnames, and other sensitive data from the report.
