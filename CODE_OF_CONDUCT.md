# Code of Conduct

MailLayers follows the Contributor Covenant Code of Conduct, version 2.1.

Be respectful, constructive, and professional in issues, pull requests, discussions, and other project spaces. Harassment, abusive language, discrimination, and sustained disruption are not acceptable.

Report conduct concerns privately to `support@maillayers.com`. Maintainers may remove, edit, or reject contributions and may temporarily or permanently restrict participation when behavior violates this policy.
